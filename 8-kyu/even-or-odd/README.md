# Even or Odd

**Difficulty:** 8 kyu

## Problem Summary

Write a function that takes an integer as input and returns `"Even"` if the number is even, or `"Odd"` if the number is odd.

## Examples

```text
Input: 2
Output: "Even"

Input: 7
Output: "Odd"

Input: 0
Output: "Even"
```

## Approach

Use the modulo (`%`) operator to determine whether the number is divisible by `2`. If the remainder is `0`, return `"Even"`; otherwise, return `"Odd"`.

## Complexity

* **Time:** O(1)
* **Space:** O(1)

## Concepts Practiced

* Conditional statements
* Modulo operator (`%`)
* Functions
* Integer arithmetic

## Notes

A beginner-friendly kata for practicing conditional logic and learning how the modulo operator can be used to determine whether a number is even or odd.

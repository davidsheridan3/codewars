# Find the Smallest Integer in the Array

**Difficulty:** 8 kyu

## Problem Summary

Write a function that takes a non-empty list of integers and returns the smallest value in the list.

## Examples

```text id="vgb6pn"
Input: [34, 15, 88, 2]
Output: 2

Input: [34, -345, -1, 100]
Output: -345
```

## Approach

Use Python's built-in `min()` function to find and return the smallest value in the list.

## Complexity

* **Time:** O(n)
* **Space:** O(1)

## Concepts Practiced

* Lists
* Built-in functions (`min()`)
* Functions
* Working with integers

## Notes

A simple kata for becoming familiar with Python's built-in functions. The `min()` function efficiently returns the smallest element in an iterable without requiring a manual loop.

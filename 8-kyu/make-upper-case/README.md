# MakeUpperCase

**Difficulty:** 8 kyu

## Problem Summary

Write a function that takes a string as input and returns the same string with all alphabetic characters converted to uppercase.

## Examples

```text
Input: "hello"
Output: "HELLO"

Input: "CodeWars"
Output: "CODEWARS"

Input: "123abc!"
Output: "123ABC!"
```

## Approach

Use Python's built-in `upper()` string method to convert all lowercase letters to uppercase and return the result.

## Complexity

* **Time:** O(n)
* **Space:** O(n)

## Concepts Practiced

* Strings
* String methods
* Functions
* Returning values

## Notes

A simple kata for becoming familiar with Python's built-in string manipulation methods. The `upper()` method creates and returns a new uppercase string without modifying the original.
